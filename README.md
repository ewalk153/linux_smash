## Linux Smash

A simple app to play sounds and show letters when the keyboard is used.

!(screenshot.png)

setup:
```bash
brew install pyqt
/usr/local/opt/python@3.8/bin/python3 -m venv venv
source venv/bin/activate
pip install -Ur requirements.txt
```

later when you come back:
```bash
source venv/bin/activate
```
